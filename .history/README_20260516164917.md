# Solar Sensor Intelligence — Demo Frontend

Business-focused frontend demo showing solar generation history, sensor health analytics, AI-backed verdicts, and automated water/cleaning cycle recommendations.

Key features
- Solar energy history and daily trends (ML-derived expectations)
- Sensor telemetry panels: signal health, confidence, noise, and thermal events
- AI health verdict with concise reasons and remediation guidance
- History-driven water-cycle recommendation plus manual "Initiate Water Cycle" action

Quick start
1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open the dashboard: http://localhost:3000

Demo notes
- Use the "Solar ML Controls" panel to run the demo flow.
- The "Initiate Water Cycle" button is in the Sensor Snapshot panel and simulates a cleaning event.

Tech stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Recharts (charts)

Integrations
- This is frontend-only: swap the mock data in `app/page.tsx` for API endpoints to connect real sensors and solar telemetry.
